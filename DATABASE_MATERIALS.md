
This project uses SQLite3 and Django to manage the database.

The tables and models are contained in the `./vmbsite/inventory/models.py` file.

The queries are throughout the `./vmbsite/inventory/views.py` file.

The API endpoints and separated in the views file from the page endpoints to modularize and improve ergonomics of the project.

The ER diagram is in the `ER_Diagram.png` file.
